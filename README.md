## 📌 Opis projektu
GamingHub to platforma / strona gamingowa stworzona w ramach pracy inżynierskiej na studia.  
Projekt prezentuje nowoczesny (jak na vanilla stack) interfejs do przeglądania gier, newsów gamingowych, multimediów i innych elementów społeczności graczy – wszystko w oparciu o czysty HTML, CSS i JavaScript.

Celem było pokazanie umiejętności budowania responsywnej, funkcjonalnej i estetycznej aplikacji webowej bez użycia frameworków oraz frameworków frontendowych.

## 🚀 Uruchamianie projektu
### 1️⃣ Klonowanie repozytorium
```bash git clone https://github.com/michalpawlowski-mp/GamingHub.git```  
```bash cd GamingHub```

### 2️⃣ Uruchomienie (brak instalacji zależności)
Projekt nie wymaga npm/yarn/pnpm – wystarczy otworzyć plik główny.

```bash open index.html```         # macOS  
```bash start index.html```         # Windows  
```bash xdg-open index.html```     # Linux  

albo po prostu kliknij dwukrotnie na plik `index.html`

## 🛠 Technologie
- HTML5
- CSS3 (w tym Flexbox, Grid, animacje, media queries)
- Vanilla JavaScript (ES6+)

Bez bundlerów, bez zewnętrznych bibliotek – 100% vanilla.
